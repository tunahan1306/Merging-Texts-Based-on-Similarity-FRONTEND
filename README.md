## Getting Started


### Installation

1. Clone the repo
   ```sh
   git clone https://github.com/tunahan1306/Merging-Texts-Based-on-Similarity-FRONTEND.git
   ``` 
2. Install NPM packages
    ```sh
    cd yazlabfrontend
    ```
    ```sh
    npm install
    ```
4. Enter your API in `src/config.jsx`
   ```js
   export const BASE_URL = "http://127.0.0.1:8080";
   ```
